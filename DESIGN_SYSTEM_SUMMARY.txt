- Smooth 300ms transitions on all interactive elements

  - 3-column summary grid:  grid grid-cols-3 gap-4

SPACING HIERARCHY:

  - Between sections: gap-6 to gap-8 (1.5rem - 2rem)

  - Border radius:    rounded-xl (12px) standard for cards

SECTION ORGANIZATION:
  - Header with title + description
  - Grid of summary cards (often 3-column)
  - Space-y-4 for vertically stacked items
  - Consistent max-width containers

  - Shadow: shadow-glow-sm (0 4px 12px)

  - Hover:  Gradient shift + scale-105

ACCENT ACTION BUTTONS:
  - Amber:  from-amber-600 to-amber-700 (Start actions, secondary CTA)
  - Green:  from-green-600 to-green-700 (Confirm, complete actions)

NAVIGATION/ICON BUTTONS:
  - Default:  text-slate-400
  - Hover:    text-slate-200 + bg-slate-800/60
  - Active:   Gradient background + accent border + text-accent
  - Icons:    Scale 110% on hover

INTERACTIVE FEEDBACK:
  - Duration:  300ms standard (transition-all duration-300)
  - Effects:   Scale (1.05), lift (-translate-y-0.5), shadow enhance
  - Disabled:  opacity-50
  - Focus:     border-accent/40 outline-none (inputs)

================================================================================
                      CARD & CONTAINER STYLES
================================================================================

STANDARD CARD (Glass Morphism):
  Element:     bg-neutral-900/50 backdrop-blur-sm
  Border:      border border-neutral-800/50
  Shadow:      shadow-[0_4px_24px_rgba(0,0,0,0.6)]
  Hover:       shadow-[0_8px_32px_rgba(0,0,0,0.8)] + -translate-y-0.5
  Duration:    transition-all duration-300

COLORED ACCENT CARD (Example: Amber):
  Background:  bg-gradient-to-br from-slate-800/70 to-slate-900/50
  Border:      border-amber-600/50 → hover: border-amber-500/70
  Shadow:      Layered (dark + accent tint + inset light)
  Glow:        Radial gradient on hover (opacity 0 → 100%)
  
NESTED/INNER CARDS:
  Style:       bg-slate-900/60 border border-slate-700/60 p-3
  Use:         For contained detail sections within main cards

PADDING & SPACING:
  - Large:    p-6 to p-8 (24-32px)
  - Medium:   p-4 to p-6 (16-24px)
  - Small:    p-3 (12px)

================================================================================
                      ANIMATIONS & TRANSITIONS
================================================================================

KEYFRAME ANIMATIONS:
  - fadeIn:        opacity 0→1 (0.5s ease-out)
  - fadeInUp:      opacity 0→1 + translateY 20px→0 (0.6s ease-out)
  - slideInRight:  opacity 0→1 + translateX -20px→0 (0.5s ease-out)
  - popIn:         scale 0.95→1.02→1 + opacity (0.3s ease-out)
  - shadow:        box-shadow pulsing (3s ease-in-out infinite)

TRANSITION PATTERNS:
  - Standard:      transition-all duration-300 cubic-bezier(0.4, 0, 0.2, 1)
  - Longer:        transition-all duration-500
  - Transform:     transition-transform duration-300

HOVER EFFECTS:
  - Scale:         hover:scale-105 (5% larger)
  - Lift:          hover:-translate-y-0.5 to -translate-y-1 (2-4px up)
  - Shadow:        hover:shadow-lg or hover:shadow-2xl
  - Color:         hover:bg-X hover:border-accent/40
  - Icon scale:    group-hover:scale-110 (10% larger)

ANIMATION ENTRANCE:
  - Use animationDelay for staggered entrance
  - Typical: 0ms, 200ms, 400ms, 600ms for sequential items
  - Creates flowing, choreographed appearance

================================================================================
                      SHADOW & DEPTH SYSTEM
================================================================================

SHADOW DEPTH LEVELS:
  - Light:     shadow-md (0 4px 6px) / shadow-glow-sm (0 4px 12px)
  - Medium:    shadow-[0_4px_24px_rgba(0,0,0,0.6)] (standard card)
  - Heavy:     shadow-2xl (0 25px 50px) / shadow-glow-lg (0 8px 32px)
  - Glow:      0 8px 32px rgba(217, 170, 239, 0.2) (accent glow)

INSET SHADOWS (Subtle emboss):
  - Small:     inset 0 1px 2px rgba(255, 255, 255, 0.05)
  - Medium:    inset 0 1px 2px rgba(255, 255, 255, 0.1)

LAYERED SHADOWS (Complex):
  - Card:      0 12px 32px rgba(0,0,0,0.3) + color tint + inset light

  - Hover:     rgba(163, 163, 163, 0.5)

  - Backdrop:  blur(8px-12px)
  - BG:        rgba(23, 23, 23, 0.4-0.6)
  - Border:    1px solid rgba(64, 64, 64, 0.2-0.5)
  - Shadow:    0 4px 24px rgba(0,0,0,0.6) + inset light

GLOW/HOVER OVERLAYS:
  - Technique: Radial gradient with opacity transition
  - Positioned: absolute inset-0 (covers entire element)
  - Triggered:  group-hover:opacity-100
  - Duration:   transition-opacity duration-500

  - lg: (1024px) for larger layouts
  - Example: grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4

  - Full width: w-full with max-w-4xl mx-auto for content

================================================================================
                  VISUAL POLISH & MICRO-INTERACTIONS
================================================================================

POLISH TECHNIQUES:
  1. Backdrop filters (blur effects) for glass morphism
  2. Radial gradient overlays on hover (glow effect)
  3. Inset shadows for subtle embossing
  4. Staggered animations for choreographed entrance
  5. Icon scaling and shine effects on hover
  6. Smooth color transitions on all interactive elements
  7. Subtle lift effect (-translate-y) on card hover
  8. Opacity-based layering for visual hierarchy

INTERACTIVE FEEDBACK:
  - Every button/card has clear hover feedback
  - Scale transformations feel responsive and alive
  - Shadow changes enhance perceived elevation
  - Color shifts indicate interactivity
  - Animations are smooth and not jarring (300ms standard)

ACCESSIBILITY CONSIDERATIONS:
  - Focus states defined (border-accent/40)
  - Disabled states have opacity-50
  - Sufficient color contrast (light text on dark)
  - Icons have aria-labels (not visible, inferred)
  - Keyboard navigation supported (standard inputs)
